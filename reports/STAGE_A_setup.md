# Stage A — 装置搭建与坐标固定

**日期:** 2026-05-18
**项目:** 力–位移解耦模型 APMD（UVA AIME 磁解耦项目）
**本轮性质:** 第一遍走流程（流程熟悉版）

---

## 1. 目的

把最终的「磁–力传感」机械结构一次性装好并固定坐标系,作为后续所有阶段
(B–P)的物理基础。原则:**装好后中途不拆**,保证各阶段数据在同一几何下可比。

## 2. 装置组成

| 部件 | 型号 / 说明 | 作用 |
|---|---|---|
| 电动测试台 | Mark-10 **Model F305**,最大载荷 300 lbF / 1.3 kN | 提供垂直压缩运动;屏幕显示位移 d |
| 力传感器 | DYLY-103-5KG S 型力传感器（计划值,实物待确认*) | 测接触法向力 F |
| 磁传感器 | Adafruit MLX90393 breakout(GAIN_1X, RESOLUTION_16) | 测三轴磁场 B=(Bx,By,Bz) |
| 微控制器 | Adafruit QT Py M0(ATSAMD21E18A),CircuitPython | 读 MLX90393,USB-CDC 输出整数 CSV |
| 样品 | 针织软体样品,磁化体嵌入其中(待确认*) | 受压同时产生力 F 与磁场变化 ΔB |
| 升降台 | 铝制剪式升降台(lab jack) | 粗调磁传感器初始高度 |

\* 标「待确认」项见第 6 节。

## 3. 装配链与坐标系

**装配链(从上到下):**
```
Mark-10 压头 / 力传感器
   ↓
磁化软体样品(磁化体嵌入软体)
   ↓
MLX90393(平放固定于白色托板)
   ↓
铝制剪式升降台 → 桌面
```

**坐标系(详见 [`stageA_setup/coordinate_definition.txt`](../stageA_setup/coordinate_definition.txt)):**
- `z` 垂直压缩方向(Mark-10 压头运动方向);`x` 导轨横向;`y` 垂直于 x,z。
- `F`(N)接触法向力,F>0 为压缩。
- `q`(mm)磁化体底面到传感器顶面间隙(纯位移阶段用)。
- `d`(mm)从接触点起算的压缩位移,接触点 d=0。
- `B=(Bx,By,Bz)`(µT),`B0` 为装好磁体未接触时的 B,`ΔB=B−B0`。

## 6. 已知问题与风险（待正式轮处理）

1. **磁传感器未刚性固定** — breakout 板目前擱于托板上,托板擱于升降台。
   重复性要求亚毫米级,正式轮需用非铁磁材料(铝/塑料)将板剛性固定。
2. **升降台柔度** — 剪式升降台有间隙与弹性形变,压缩阶段反作用力经其传递。
   缓解:高度调好后锁死,精确 q/d 全部由 Mark-10 控制,升降台只作粗调。
3. **力通道方案待确认** — 压头下银色金属块的力信号疑似经螺旋线进入 Mark-10
   屏幕,与决策点 A「Mark-10 不测力、力走独立 Arduino Uno」不一致。该项决定
   `continuous_collect.py` 的串口架构,须在 Stage D(接触点)前澄清。
4. **样品/磁体待确认** — 磁化体是否确嵌于针织样品内、是否与传感器同轴;针织
   体偏软,压缩重复性可能偏弱。
